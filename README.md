# ------- Ai Business Analyzer --------  #

## Description
AI Business Analyzer is an advanced web platform tool designed to help businesses analyze their operations, market trends, and customer behavior using artificial intelligence. It provides insights and recommendations to improve business strategies and decision-making processes.


## Tech Stack
- Backend: Python, Flask
- Frontend: HTML, CSS, JavaScript
- Database: MySQL
- AI/ML: Keras in TensorFlow , Scikit-learn
